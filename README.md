## 📌 Project Overview

This project demonstrates web scraping using Python. Data was extracted from the **Books to Scrape** website using the **Requests** and **BeautifulSoup** libraries. The scraped data was cleaned, analyzed, and saved as a CSV file.

---

## 🎯 Objective

- Scrape book information from a website.
- Store the extracted data in a structured format.
- Perform basic data cleaning.
- Conduct Exploratory Data Analysis (EDA).
- Export the cleaned dataset.

---

## 🛠️ Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset Fields

- Title
- Price
- Availability
- Rating

---

## 📊 Project Workflow

1. Import required libraries
2. Send HTTP request
3. Parse HTML using BeautifulSoup
4. Extract book details
5. Scrape multiple pages
6. Create a Pandas DataFrame
7. Save the dataset as CSV
8. Clean the data
9. Perform Exploratory Data Analysis (EDA)
10. Visualize the data using charts

## 📈 Analysis Performed

- Dataset information
- Missing value check
- Duplicate value check
- Price cleaning
- Statistical summary
- Rating distribution
- Price distribution
- Average price by rating
  
## ✅ Result

Successfully scraped book information from all 50 pages of the **Books to Scrape** website, cleaned the dataset, performed basic data analysis, and exported the results into CSV format.
